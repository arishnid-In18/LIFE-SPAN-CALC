Life Span Calculator & Real-Time Calculator

A responsive web application built using HTML, CSS, and JavaScript (DOM Manipulation) that calculates a user's age and life statistics based on their name and date of birth. The application displays the total years, months, days, hours, minutes, and seconds the user has lived. It also includes a built-in real-time calculator for performing everyday arithmetic operations.

✨Features

Enter your name and date of birth
Calculate and display:
Age in years
Total months lived
Total days lived
Total hours lived
Total minutes lived
Total seconds lived
Dynamic updates using JavaScript DOM manipulation
User-friendly and responsive interface
Integrated real-time calculator for quick calculations
Clean and beginner-friendly code structure

This project combines a Life Span Calculator and a Real-Time Calculator into a single interactive web application.
